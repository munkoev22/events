Пробный ЕГЭ по Математике

Институт математики и информатики БГУ приглашает учащихся 11-х классов школ и выпускников СПО г. Улан-Удэ и республики Бурятия принять участие в пробном экзамене по математике профильного уровня (ЕГЭ по математике).

Варианты пробного экзамена и их проверка будет осуществляться действующими экспертами экзаменационной комиссии по математике.

Пробный ЕГЭ по Информатике

Институт математики и информатики БГУ приглашает учащихся 11-х классов школ и выпускников СПО г. Улан-Удэ и республики Бурятия принять участие в пробном экзамене по информатике.

Пробный ЕГЭ будет проводится на компьютерах с использованием тестирующей системы. Поддерживаемые языки программирования: C++, Pascal, Python. Среды разработки: CodeBlocks, PascalABC, Python IDLE.

В случае дистанционного участия участники должны иметь доступ к сети Интернет, браузер (Mozilla, Chrome, Edge) и необходимый пакет программ для работы (среда разработки, MS Office или аналог).

Подробности по телефонам: 21-97-57, 68-11-14.
